## システム概要

Quest Boardは以下の技術スタックで構築されています：

- **フロントエンド**: React 18 + TypeScript + Material-UI
- **バックエンド**: Node.js + Express + TypeScript
- **データベース**: PostgreSQL
- **キャッシュ**: Redis
- **検索**: Elasticsearch
- **コンテナ**: Docker + Docker Compose
